# 绿意盎然 - 植物生长记录桌面工具

一款清新森系风的本地植物生长记录桌面工具，采用 **Tauri + Vue3 + Element Plus** 技术栈构建。

## 🌿 特色

- **森系设计**：草木绿 + 米白 + 浅褐的自然配色，融入树叶、花朵等自然元素
- **轻量高效**：Tauri 架构保证应用体积小、启动速度快、内存占用低
- **本地存储**：数据保存在本地，隐私安全
- **系统集成**：支持系统托盘、开机自启动、桌面通知提醒

## ✨ 功能

- 📋 **植物档案管理**：添加、编辑、删除植物信息，支持标签分类
- 📷 **生长照片**：上传植物生长照片，记录成长瞬间
- 💧 **养护记录**：浇水、施肥、修剪、换盆等养护记录
- 🔔 **生长提醒**：可重复的养护提醒，支持桌面通知
- 📚 **养护知识**：内置植物养护知识库，支持自定义添加
- 📊 **数据统计**：多样化图表展示养护数据和生长趋势
- ⚙️ **数据管理**：支持数据导入导出，备份方便

## 🚀 快速开始

### 前置要求

- Node.js >= 16
- Rust 环境
- WebView2 (Windows 10/11 已内置)

### 开发运行

```bash
# 安装依赖
npm install

# 启动 Web 开发服务
npm run dev

# 启动 Tauri 桌面应用开发
npm run tauri:dev
```

### 构建发布

```bash
# 构建桌面应用
npm run tauri:build
```

构建产物将输出到 `src-tauri/target/release/bundle` 目录。

│   ├── icons/           # 应用图标
│   └── tauri.conf.json  # Tauri 配置
├── public/              # 静态资源
├── index.html           # HTML 入口

## 📝 数据存储

应用数据保存在浏览器 localStorage 中，包括：

- 植物档案信息
- 养护记录
- 照片（Base64 格式）
- 提醒事项
- 养护知识库
- 应用设置

> 建议定期在「设置」页面导出数据进行备份。

## 🪴 默认示例数据

首次启动时会自动创建示例数据，包括：
- 3 棵示例植物（绿萝、桃蛋多肉、薄荷）
- 若干养护记录和提醒
- 4 篇养护知识文章

## 🔧 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI 组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **图表**：ECharts + vue-echarts
- **构建工具**：Vite
- **桌面框架**：Tauri 1.x
- **后端语言**：Rust
